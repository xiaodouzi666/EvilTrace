Self-correction policy:

For every candidate finding:

1. Identify supporting artifacts.
2. Identify contradictory artifacts.
3. Check whether every entity appears in the evidence graph.
4. Check whether conclusion language is stronger than evidence supports.
5. If unsupported, reject.
6. If partially supported, downgrade to inferred.
7. If contradicted, re-plan targeted investigation.
8. If tool failed, try an alternate MCP tool.
9. If max_iterations is reached, mark unresolved claims as needs_review.
